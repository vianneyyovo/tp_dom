  1. Core DOM - modèle standard pour tous les types de documents
  2. XML DOM - modèle standard pour les documents XML
  3. HTML DOM - modèle standard pour les documents HTML
Cest quoi le HTML DOM?

Le DOM HTML est un modèle d'objet standard et une interface de programmation pour le HTML. Il définit

  • Les éléments HTML en tant qu'objets
  • Les propriétés de tous les éléments HTML
  • TLes méthodes d'accès à tous les éléments HTML
  • Les événements pour tous les éléments HTML
